# Check for clones

First of all, I start by investigating if any two (or more) of the sporocarps in my dataset belong to the same genet. 

This is done in poppr in the R Notebook. 96 genetic individuals are identified, which matches the number of sporocarps. 

Pairwise genetic distances were calculated and are plotted below. There are a few pairwise distances which are quite low and maybe I should check on them a bit closer. 

![image](https://github.com/LydiaT12/Amanita-muscaria-population-genetics/assets/124113370/559db434-d7eb-4a47-8445-4a95e39fa04e)

I'm currently investigating this by looking at some different distance measures. It would also be a good idea to figure out which indiviudals are in that tail and look at them. In particular, could I assume a specific relationship between them?
