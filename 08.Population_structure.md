## fastStructure
<!-- https://rajanil.github.io/fastStructure/ -->
Basic bit of code is:
```
structure.py -K 2 --input ../04.PLINK/scaffold1_bi_noind_bed --output scaffold1_K2
```

I'm going to want to do this in a loop, trying different values of K

```
for k_val in {1..9}; do
structure.py -K ${k_val} --input ../04.PLINK/scaffold1_bi_noind_bed --output scaffolds1to28;
done 
```

Having tested for a bunch of values of K, we test which one is optimal:

```
chooseK.py --input=scaffolds1to28
````
Model complexity that maximizes marginal likelihood = 1
Model components used to explain structure in data = 1

Sooo... no grouping.
