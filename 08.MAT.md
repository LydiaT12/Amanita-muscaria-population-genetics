Poking around in this file, I see that some (but not all) of the heterozygous loci have phasing information. The attribute PGT records physical phasing with a character like 0|1 to denote which read is the alternate. PID is a phasing ID, looking like 508297_A_G. PS is a phasing set, typically the base of the first phased allele in the set. I think that I should filter by having a PS attribute and then see how easy it is to cluster these groups. 

```
gatk VariantFiltration  -R ../1.ref_genome/Amanita_muscaria_genome_2024_masked.fasta  -V F1.g.vcf.gz  --filter-name "PS" --filter-expression "vc.hasAttribute("PS")" -O HD_F1.filtered.g.vcf.gz
```
