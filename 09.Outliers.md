## Check for homozygosity

### Runs of homozygosity

BCFtools has a ROH plug-in

Make a file with allele frequencies
```
bcftools query -f'%CHROM\t%POS\t%REF,%ALT\t%AF\n' ../../02.Data_quality/6.All_datasets/filtered_dataset_noclone.vcf.gz | bgzip -c > filtered_noclone_AFs.tab.gz
tabix -s1 -b2 -e2 filtered_noclone_AFs.tab.gz
echo '##INFO=<ID=AF1KG,Number=1,Type=Float,Description="Allele Frequency">' > filtered_noclone_AFs.tab.gz.hdr
```

Then run the scripts 
```
bcftools roh --AF-file ROHin/filtered_noclone_AFs.tab.gz ../../02.Data_quality/6.All_datasets/filtered_dataset_noclone.vcf.gz -o testout.txt
```
