# Raw data

The mitogenome variants were called. The SNP quality filtering which was applied to the nuclear genome results in no mitogenome variants, so I stick with just the default GATK filters. I use only SNPs. The dataset is in both bgz and gz compression formats

```
02.Data_quality/mitogenome_SNP.vcf.bgz  02.Data_quality/mitogenome.vcf.gz
```

I need/want a plink dataset to match

```
plink2 --vcf ../02.Data_quality/mitogenome.vcf.gz --allow-extra-chr --make-pgen --out mitogenome
plink2 --pfile mitogenome --allow-extra-chr --max-alleles 2 --make-bed --out mitogenome_no_multiallelic
```
