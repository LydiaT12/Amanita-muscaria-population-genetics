## VCFtools

Calculating LD for all pairs of SNPs is a big dataset, and I won't use most of it. I'm going to subset the data to one scaffold at a time, and calculate LD just within that scaffold. If I want to improve my genome later, I can come along and calculate LDs across the ends of pairs of scaffolds - but that will still be a smaller problem, and the initial scaffold-subsetted data will tell me over what distance I can expect to see stuff.

I'll just do the calculations over the first 132 scaffolds (everything below the mitogenome). I use a for loop to do calculations for each scaffold in turn

```
for ii in {1..132}
do
# first subset to scaffold ii
vcftools --gzvcf filtered_dataset.vcf.bgz --chr Scaffold${ii}_Amanita_muscaria  --recode -c --recode-INFO-all | gzip -c > 5.LDcalcs/scaffold${ii}.vcf.gz
# then calculate genotype disequilibriums
vcftools --gzvcf 5.LDcalcs/scaffold${ii}.vcf.gz --geno-r2 -c > 5.LDcalcs/scaffold${ii}.geno.ld
done
```

## Calculating recombination rates

I can estimate the decay of r^2 as a function of population recombination. I use only close variants, since this calculation uses the assumption that there won't be multiple recombination events between a pair of variants. I use the population-specific r^2 values and subset to just those sites less than 1000bp apart. This is still a lot of data. I do this one scaffold at a time (only th first 28 scaffolds bc it's too much) and write the output file.
