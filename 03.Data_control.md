For some analyses I will want datasets which comprise a single scaffold. 

```
vcftools --gzvcf scaffolds_1to28_filtered.vcf.gz --chr Scaffold1_Amanita_muscaria  --recode -c --recode-INFO-all | gzip -c > 1.subset_files/test_scaffold1.vcf.gz
```
The -recode-INFO-all tag is needed to keep important metadata

```
for ii in {1..28}; do vcftools --gzvcf scaffolds_1to28_filtered.vcf.gz --chr Scaffold${ii}_Amanita_muscaria  --recode -c --recode-INFO-all | gzip -c > 1.subset_files/test_scaffold${ii}.vcf.gz ; done
```

# Filter stuff out

What do I want to filter? I already did the very default default settings of GATK's default filtering. 

I want to filter out anything that looks like it might be an error. To do this, I need to identify outliers. The first things to check are read depth and mapping quality. These give us an idea of how much we trust the output from the haplotype calling, and may suggest areas in the reference genome assembly which are messy. 

Area of concern 1: Indels are more likely than SNPs to be errors [where did I read that?]. Multiallelic sites are not well supported by some downstream software. What overall effect do they have on the data?
I compare on both supercontigs 2 and 3, since they have quite different mapping rates/quality scores. Overall, indels and multiallelic sites appear to be a minority of variants. 

Some of the highest variant-per-site scores are reduced by this filtering. I suspect that this is mostly due to multiallelic sites, where the existance of multiple variants means that the major variant has less potential to be common. Spikes in variants-per-site associated with spikes in read depth or quality don't appear to be disproportionately affected by the filtering.

## Do any sensible vcf filtering

Some downstream analyses don't like multiallelic sites. Indels are more likely than SNPs to be errors. Can filter both out. 
```
vcftools --gzvcf 1.subset_files/test_scaffold1.vcf.gz --max-alleles 2 --recode -c --recode-INFO-all | gzip -c > 1.subset_files/scaffold1_biallelic.vcf.gz
vcftools --gzvcf 1.subset_files/test_scaffold1.vcf.gz --remove-indels --recode -c --recode-INFO-all | gzip -c > 1.subset_files/scaffold1_noindels.vcf.gz
vcftools --gzvcf 1.subset_files/scaffold1_biallelic.vcf.gz --remove-indels --recode -c --recode-INFO-all | gzip -c > 1.subset_files/scaffold1_biallelic_noindels.vcf.gz
```
