# Check mapping depth, quality, etc

Get the mean read depth for each site (averaged across all individuals). These two do the same thing, but the -c > method lets me direct the output to a particular folder. 

```
vcftools --gzvcf 1.subset_files/test_scaffold1.vcf.gz --site-mean-depth --out scaffold1
vcftools --gzvcf 1.subset_files/test_scaffold1.vcf.gz --site-mean-depth -c > 2.readstats/test_scaffold1.ldepth.mean
```
