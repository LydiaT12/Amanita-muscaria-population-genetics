## Rooting the phylogeny

I want to build a tree using only variants which exist in the outgroups, solely to decide where to root the A. muscaria phylogeny. I'd rather do this as a seperate step to the main tree bulding because there's so little overlap in the sites involved. 

First, I take the intersection of the outgroup file and the invariant-inclusive muscaria file

```
bcftools isec -p rootingvars outgroups.vcf.gz ../../../02.Data_quality/8.Invariants/invariants_1to28.noclone.vcf.bgz
```

```0002.vcf  0003.vcf``` contain the records in one file shared by the other. I merge these. 

```
bgzip 0002.vcf
bgzip 0003.vcf
bcftools index 0002.vcf.gz
bcftools index 0003.vcf.gz
bcftools merge 0002.vcf.gz 0003.vcf.gz -Oz -o rootingvars.vcf.gz
```

Now convert to phylip and run iqtree
```
../../vcf2phylip.pl --input rootingvars.vcf.gz
iqtree2 -s rootingvars.min4.phy -st DNA -m GTR+G -bb 1000  -pre rootingfile
```
