# Check for clones and close relatives

## Poppr

First of all, I start by investigating if any two (or more) of the sporocarps in my dataset belong to the same genet. If they do, then further analyses need to allow for the fact that some genotypes are represented multiple times. 

This is done in poppr in the R Notebook. 96 genetic individuals are identified, which matches the number of sporocarps. No repeated genotypes are identified. 


Now I calculate pairwise genetic distances. Abnormally small distances are the pairs
> T3 and T4
> 
> H3 and H6
>
> F2 and F9
>
> F4 and F11
>
> T10, T11 and T18

Each pair of sporocarps with an exceptionally small genetic distance exists within a single site, which fits with expectations. They might be clones with variation in the sequencing/variant calling. They might be close relatives.

./king -b ../04.PLINK/testpb.bed --related --degree 2 --prefix testpb > testpb.out
