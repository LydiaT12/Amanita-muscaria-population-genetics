# Check for clones

## Clone correction

First of all, I start by investigating if any two (or more) of the sporocarps in my dataset belong to the same genet. If they do, then further analyses need to allow for the fact that some genotypes are represented multiple times. 

This is done in poppr in the R Notebook. 96 genetic individuals are identified, which matches the number of sporocarps. No repeated genotypes are identified. 

## Pairwise genetic distances

Pairwise genetic distances (Hamming metric, on the variants) were calculated and are plotted below. There are a few pairwise distances which are quite low. These could be parent/child or sibling relationships (or something of the like), but they could still potentially be from the same genet with a few extra mutations seperating them. I had better check a bit closer before claiming that there are no repeated genets in the dataset.
