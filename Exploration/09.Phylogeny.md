I have three (potential) outgroup genomes in fasta format. I want to compare these with my genome assembly and create vcf files which can be combined with my main dataset.

```

## Outgroup
Genome of Amanita phalloides (SRR10129174) reads downloaded from NCBI https://trace.ncbi.nlm.nih.gov/Traces/?view=run_browser&acc=SRR10129174&display=download

Reads mapped to genome. Then sort it. 

```
module load SAMtools
samtools sort -o outgroup_phalloides.sorted.bam mapped_reads_phalloides.bam 
```

quick and easy variant calling

```
module load BCFtools
bcftools mpileup -O b -o outgroup_phalloides.bcf -f ../../1.ref_genome/Amanita_muscaria_genome_2024.fasta outgroup_phalloides.sorted.bam
bcftools call --ploidy 1 -m -v -o outgroup_phalloides.vcf outgroup_phalloides.bcf
# vcfutils.pl varFilter outgroup_phalloides.vcf  > outgroup_phalloides.filtered.vcf # I didn't do this
